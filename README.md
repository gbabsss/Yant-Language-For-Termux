# Yant-Language-For-Termux

This Is A Language For Termux That Uses Python 3.1+
The Yant Language Files Are End With .y You Can Use yant (filename.y) for now syntax is limited 2-3 functions available you can find example .y file in repo

# Yant-Install

For Installing Yant Language You Have Two Options

# 1. Option

Installing With Python Just Download Yant_Installer.py And Paste This Command To Terminal

**python yant_installer.py**

# 2. Option 

On This Option You Can Use .sh File Just Paste This Command To Terminal

**./yant_installer.sh**

And You Were Done You Can Use yant (filename.y) For Your Yant Language Files

*Reminder; This Is Only For Termux Not For Other Linux Stuff There Were Added In Other Updates Stay Tuned!*
